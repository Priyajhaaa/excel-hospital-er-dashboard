# 🚑 Hospital Emergency Room Dashboard

## 📌 Project Overview

This project presents an interactive Hospital Emergency Room Dashboard built in Microsoft Excel using real-world healthcare data. The dashboard helps analyze patient flow, waiting time, satisfaction scores, department referrals, and admission trends through dynamic visualizations and KPI cards.

The main objective of this project is to transform raw healthcare data into meaningful insights that support operational efficiency and data-driven decision-making.

---

# 📊 Project Objectives

- Analyze emergency room patient data
- Monitor average waiting time
- Evaluate patient satisfaction trends
- Identify peak emergency hours
- Visualize department-wise referrals
- Compare admission and non-admission rates
- Build an interactive healthcare analytics dashboard

---

# 📁 Dataset Information

The dataset contains the following information:

- Patient ID
- Admission Date
- Gender
- Age
- Race
- Wait Time
- Satisfaction Score
- Department Referral
- Admission Status

# 📊 Key Insights

- Peak patient traffic was observed during evening hours
- Average waiting time increased during high admission periods
- Most referrals were directed to General Medicine
- Patient satisfaction varied across departments
- Admission trends helped identify workload distribution
